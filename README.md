# A collection of sorting algorithms implemented in C++

## Bubble Sort
A slow sorting algorithm for the simplest data sets

| Case  | Performance |
| :---: | :---: |
| Worst case performance   | O(n^2)  |
| Best case performance  | O(n)  |
| Average case performance  | O(n^2)  |
| Space  | O(1)  |
